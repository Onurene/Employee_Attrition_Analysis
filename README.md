# Employee_Attrition_Analysis
This project focuses on analysing various factors responsible for employee attrition. 

## Analysis is based on the categorical variables like
1. Business Travel
2. Department
3. Education
4. Gender
5. Marital Status
6. Age
7. Job role
8. Over time

## Techstack used
1. Databricks 
2. SQL
3. Visualization
